User Types

Includes classes, structures, enumerations, unions, and typedefs. Use CamelCase with an initial upper-case letter.

Ex. class Vec2; class ComboBox; union BytePack;

Variables

All variables use camelCase with an initial lower-case letter. Global variables are prefixed with "g". Member variables are prefixed with "m". Parameters are prefixed with "p".

Ex. const int gAppVersion; float mRotation; Vec2 pPosition; int count;

Namespaces

All namespaces use camelCase with an initial lower-case letter. Note that namespaces should be kept as short as possible. Usually they are represented as an acronym or abbreviation.

Ex. namespace t3d { }

Macros

All macros use ALL_CAPS_SNAKE_CASE. Note that macros are only to be used for redundant tasks that can't easily be divided into a real routine. TODO move non-naming content elsewhere
